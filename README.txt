# SQL Exercises Interactivo

Proyecto web con ejercicios interactivos para practicar consultas SQL.  
Permite escribir consultas, verificar respuestas y mostrar la respuesta correcta si es necesario.

## Cómo usarlo

1. Selecciona una categoría y subcategoría de ejercicios.  
2. Escribe tu consulta SQL en el área de texto.  
3. Haz clic en **Verificar** para comprobar si tu respuesta es correcta.  
4. Si tienes dudas, haz clic en **Mostrar respuesta** para ver la consulta correcta.

---

## Importante: Formato Sensible

- La evaluación de la consulta **es sensible a espacios y mayúsculas/minúsculas** según la consigna.  
- Asegúrate de respetar el formato correcto, incluyendo mayúsculas y minúsculas, para que la verificación funcione correctamente.  
- Aunque el sistema intenta comparar resultados y estructuras, en algunos ejercicios la consulta debe coincidir exactamente con la solución correcta.
